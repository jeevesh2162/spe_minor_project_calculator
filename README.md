# Scientific Calculator with DevOps

A mini-project for CS 816 - Software Production Engineering.

## Tech Stack
- **Backend**: Java 17, Spring Boot, Maven, JUnit
- **Frontend**: React, Vite, Framer Motion, Lucide Icons
- **DevOps (Pending)**: Docker, Jenkins, Ansible, ELK

## Prerequisites
- Java 17+
- Maven
- Node.js & npm

## Running the Backend
1. Navigate to `calculator-backend`
2. Run `mvn spring-boot:run`
3. API is available at `http://localhost:8080/api/calculator`

## Running the Frontend
1. Navigate to `calculator-frontend`
2. Run `npm install`
3. Run `npm run dev`
4. Access via `http://localhost:5173`

## Features
- **Square Root (√x)**
- **Factorial (x!)**
- **Natural Log (ln x)**
- **Power (x^b)**
- **Unit Testing** (JUnit)
- **Logging** (SLF4J/Logback)
